# Furniwell Indonesia — Phase 1 Local Prototype

This folder is a clean static prototype built from the supplied WordPress backup and the Phase 1 execution document.

## How to preview

Option A: double-click `index.html`.

Option B (recommended): open Command Prompt in this folder and run:

    py -m http.server 8000

Then open `http://localhost:8000`.

## What was completed

- Modern responsive homepage.
- About page.
- Our Factory & Process page using original workshop/shipment images.
- Quality & Legality verification-status page.
- Products and project-reference page.
- Shipping & Trade Terms draft.
- Returns & Claims policy structure.
- Contact/RFQ page with office, showroom, and production separated.
- Two expertise articles.
- Lightweight CSS/JS; no external framework required.
- Selected images converted to WebP and resized.

## Claims intentionally NOT published as facts

- SVLK certificate number or active certification.
- Kiln-dried timber at 6–8% moisture content.
- Lacey Act / PPQ 505 service capability.
- Current MOQ, monthly production capacity, lead time, payment terms, and Incoterms.
- Approved testimonial text.
- Primary official NAP address.

These require evidence or management confirmation before go-live.

## Important

This folder does not modify the supplied WordPress backup and is not connected to the live website.
